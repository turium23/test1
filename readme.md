## üìç Overview

The AutoDoc-ChatGPT project is a console application that uses a ChatGPT model to generate code documentation. It takes in a code file as input and uses the AutoDoc module to generate comments for each part of the code. The resulting documentation is then added as comments to the code file itself. This project aims to automate the process of code documentation, saving developers time and effort while ensuring that their code is well-documented. Its value proposition is to provide a convenient and efficient solution to improve code quality and maintainability through autogenerated documentation.
